## ✨ Overview

This is a **simple and interactive User Registration System** built using **HTML, CSS, and JavaScript**.  
It demonstrates how frontend applications can handle user input, validation, and local data storage without a backend.

> ⚠️ This is a **demo project** created for learning and showcasing purposes only.

---

## 🎯 Features

✅ Clean and modern UI design  
✅ Smooth user experience  
✅ Form validation (username & password)  
✅ Local storage data saving (browser-based)  
✅ Instant success alert feedback  
✅ Auto-refresh after registration  
✅ Fully responsive layout  

---

## 🖥️ Tech Stack

- 🌐 HTML5  
- 🎨 CSS3 (Gradient UI + Styling)  
- ⚡ JavaScript (DOM + Local Storage)

## ⚙️ How It Works

1. User enters **username & password**
2. Form validation checks:
   - Empty fields
3. Data is stored in **browser localStorage**
4. Success alert is shown
5. Page refreshes automatically
